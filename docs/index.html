quick start

sunny — setup
# clone the repo
$ git clone https://github.com/nosey-dewdrop/sunny.git
$ cd sunny/sunny

# add your WeatherAPI key
$ cat > sunny/Config/Secrets.swift << 'EOF'
enum Secrets {
    static let weatherAPIKey = "YOUR_KEY"
}
EOF

# open in xcode and run
$ open sunny.xcodeproj
# select your device or simulator → Cmd+R

why i built this

i'm a cancer survivor. sunscreen isn't optional for me — it's survival. but even knowing that, i'd forget to reapply constantly. every weather app tells you the temperature, none of them actually remind you to protect your skin.

so i built sunny. it's simple on purpose: open the app, see the weather with a cute illustration, get reminded when it's hot enough to burn. the hand-drawn backgrounds are made in Procreate because i wanted it to feel warm and personal, not clinical.

this is my first iOS app. i learned SwiftUI, CoreLocation, and push notifications from scratch to build it. every illustration, every line of code, every design decision came from wanting something that felt like it was made by a human who actually cares whether you get sunburned.

— damla
